# 📊 Data Science Portfolio

A collection of Data Science and Machine Learning projects implemented in Python using Jupyter Notebooks. This repository showcases hands-on experience with real-world datasets, covering topics from regression and classification to deep learning and anomaly detection.

---

## 🗂️ Projects

### 1. 🏠 [California Housing Regression](./California_Housing_Regression.ipynb)
Predicting California housing prices using the California Housing dataset. Explores feature engineering, data visualization, and applies regression models to estimate median house values based on geographic and demographic features.

- **Techniques:** Linear Regression, Feature Engineering, Data Visualization
- **Dataset:** California Housing Dataset (sklearn)

---

### 2. 🚢 [Titanic Survival Analysis](./Titanic_Survival_Analysis.ipynb)
Classic binary classification problem to predict passenger survival on the Titanic. Includes exploratory data analysis (EDA), handling missing values, and building classification models.

- **Techniques:** EDA, Classification, Feature Engineering, Missing Value Handling
- **Dataset:** Titanic Dataset

---

### 3. 🔢 [MNIST Digit Recognition](./MNIST_Digit_Recognition.ipynb)
Building a deep learning model to recognize handwritten digits using the MNIST dataset. Implements neural networks and evaluates model accuracy on test data.

- **Techniques:** Neural Networks, Deep Learning, Image Classification
- **Dataset:** MNIST (Handwritten Digits)

---

### 4. 🚨 [Anomaly Detection with Autoencoder](./Anomaly_Detection_Autoencoder.ipynb)
Implementing an Autoencoder neural network to detect anomalies in data. Uses reconstruction error as a metric to identify outliers.

- **Techniques:** Autoencoders, Unsupervised Learning, Anomaly Detection, Deep Learning

---

### 5. 🔍 [Local Code Search Engine](./LocalCodeSearchEngine.ipynb)
Building a local code search engine that allows semantic search across a codebase. Leverages NLP techniques to find relevant code snippets based on natural language queries.

- **Techniques:** NLP, Semantic Search, Embeddings, Information Retrieval

## 🚀 Getting Started

### Prerequisites
Make sure you have Python 3.8+ installed along with Jupyter Notebook or JupyterLab.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ashishat404/Data-Science.git
   cd Data-Science
   ```

2. **Install required dependencies:**
   ```bash
   pip install numpy pandas matplotlib seaborn scikit-learn tensorflow keras jupyter
   ```

3. **Launch Jupyter Notebook:**
   ```bash
   jupyter notebook
   ```

4. Open any `.ipynb` file and run the cells to explore the projects!
